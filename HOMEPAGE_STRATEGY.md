### 3. The Middle Option Bias

Most people pick the middle tier. Make it the best value:

| Tier | Price | Purpose |
|------|-------|---------|
| Starter | $29 | Anchor (looks cheap) |
| **Pro** | $79 | Target (best value, "POPULAR" badge) |
| Business | $199 | Makes Pro look reasonable |

### 4. What to Include in Each Tier

**Starter ($29) â€” The Toe-Dip**
- Enough to try (100 min = ~33 calls)
- One number, one assistant
- All core features (no crippling)
- Goal: Get them in, upgrade later

**Pro ($79) â€” The Sweet Spot**
- Enough for real use (500 min = ~167 calls)
- Multiple numbers/assistants
- Calendar integration (sticky feature)
- Goal: Most customers land here

**Business ($199) â€” The Anchor**
- For power users
- API access (developers)
- Custom integrations
- Goal: Make Pro look cheap, capture whales
